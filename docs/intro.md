# Tutorial Intro

Let's discover **Everscale in less than 5 minutes**.

## Getting Started

Get started by **creating a new contract**.

## Generate a new project

```shell
mkdir my-project && cd $_
npm init --force
npm install --save tondev
```

## Generate a new contract

```shell
npx tondev sol create App
```
